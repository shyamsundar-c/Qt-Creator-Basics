Qt Quick Controls & Animations – QML Collection
A structured collection of Qt 6 QML UI components, layouts, dialogs, animations, navigation modules, and graphical effects developed as a hands-on learning and reference project.

This repository demonstrates practical implementation of:
1. Qt Quick Controls (Button, ComboBox, Slider, etc.)
2. Layout Management (Row, Column, Grid, Stack)
3. Dialogs and Navigation (StackView, SwipeView)
4. Property & Transition Animations
5. Qt6 Effects and Shader examples
6. Resource handling (.qrc) with CMake
7. Light/Dark theme switching

Built using Qt 6, Qt Quick, QML, and CMake.
Ideal for students and developers learning modern cross-platform GUI development with Qt Quick.
